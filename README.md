# Aplicativo de tarefas - TODO List

## Para rodar o projeto

Após clonar o projeto abra o terminal e use um dos comandos:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra o endereço [http://localhost:3000](http://localhost:3000) no seu navegador.

## Learn More
